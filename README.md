Die App läuft komplett im Browser, benötigt keine Installation und funktioniert auch ohne Internetverbindung.

## Deutsch

**Flip7 Score** ist ein einfacher, offline-fähiger Punktezähler für das Kartenspiel Flip7.

### Funktionen
- Bis zu 18 Spieler
- Dynamische Spielernamen (optional)
- Einzelne Spieler entfernen (auch mittendrin mit automatischer Neunummerierung)
- Offizielle Flip7-Wertung mit Live-Vorschau
- Runden-Historie pro Spieler (mit Löschfunktion)

- Vollständige Unterstützung von 12 Sprachen

1. Lade die Datei `punktezähler.html` herunter
